GOURMET GAUNTLET

My contributions:

  • Developed the player and their abilities
  • Designed all current levels
  • Developed the enemy spawning system

Gourmet Gauntlet is a twin-stick, top-down, bullet hell shooter which is currently being developed for my senior project. It was inspired by Root Rampage which was a project I developed for a game jam. In Gourmet Gauntlet you play as a chef who created a recipe to create sentient food. A rival chef, jealous of your creations, steals your recipe and your pet corndog and now you are on a mission to get them back. In the game you will have to progress through many differently food themed worlds on order to accomplish your goal and to save your pet. Each world features many levels and a unique boss at the end of each level. While this game is currently unfinished we anticipate a release in the summer of 2024.

  • Game Engine: Godot
  • Year: 2023-2024
  • Team Size: 4
  • Created For: Class Project

ROOT RAMPAGE

My contributions:

  • Developed player mechanics
  • Designed system for developing levels
  • Created 14 levels

Root Rampage is a twin-stick, top-down, bullet hell shooter which was created for the Global Game Jam 2023. After the game jam we expanded on it further and polished the game so that we could present the more finalized version of the game at Shawnee Game Conference. In Root Rampage, players play through 16 unique levels with increasing difficulty. There are multiple types of enemies each with different behavior and attack patterns. There are two unique boss fights which provide a greater challenge than the common enemies. Each level is short but challenging, encouraging players to experiment with different strategies.

  • Game Engine: Godot
  • Year: 2023
  • Team Size: 3
  • Created For: Global Game Jam 2023

ZOMBONK

My contributions:

  • Developed the enemies and their pathfinding
  • Developed the player mechanics and hammer
  • Created dramatic lighting to make the game more suspenseful
  • Created spatial sound effects to increase immersion

ZomBonk is a survival horror game created in 72 hours game jam for the Ludum Dare 50, the theme of which was "Delay the Inevitable". In ZomBonk, you are trapped in a maze that doubles as a graveyard. As time passes zombies will rise from their tombstones, you can bonk them to send them back from wence they came but if they amke it out all you will be able to do is stun them. As time passes your flashlight will also dim, while you can collect batteries to keep your flashlight on you are only delaying your fate.

QUICK SWAP CAVERNS

My contributions:

  • Developed a system for creating enemies
  • Created all 7 enemies present in the game

Quick Swap Caverns is a hack and slash game made for the Global Game Jam 2021, which had the theme "Lost & Found." The games gimmick was that every you killed an enemy you would lose your current weapon and get a new one from the enemy you killed. The game has 5 unique levels, each introducing unique enemies and greater challenges than the last.

  • Game Engine: Godot
  • Year: 2021
  • Team Size: 8
  • Created For: Global Game Jam 2021
